Hw1-Part 1: Wireframing

Here are some of the screen mockups for my possible final project HeadHunter.

Headhunter is an job hunting application inspired by the dating app Tinder. Users will display their skills and interested positions. Then the application will search through the available users and attempt to match you. Then the user will swipe left or right to select or dismiss the other member. If two users have show interest in each other then the application will create a message thread between them so they can converse.

First you have the login page. It is a simple pair of email and password fields along with two buttons. One is for signing up and the other is to log in.

Next we have the signup page. It is a email input field along with two password fields to confirm the user's input is correct. Finally there is the submission button at the bottom

Here we have our Profile page a user can provide their name, location and position of interest. Also they can write a blurb, provide a profile picture and list their skills using the tag lines.

At the top are navigational buttons, inbox will take you to the inbox buttons while matches takes you to the matching page

This is the matching page. It will display potential matches listing their information. If you are interested in contacting them you swipe to the right else swipe left to dismiss them for another user.

If two members select eachother a message thread will open between them which is displayed in the inbox.
